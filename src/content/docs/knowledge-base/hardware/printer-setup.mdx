## <AppleIcon /> Installation Instructions for macOS

### Launch and setup the Application

1.  Open Finder > Applications or use Spotlight Search (Cmd + Space) to find the Universal Print application.
2.  Open the application.
3.  When the app opens for the first time, you will need to give it permissions. Click Allow and then reopen System Settings.
4.  Go to the application in System Settings and sign-in with your VALD M365 account. If you have platform SSO configured this should happen automatically.

### Find and Install a Printer

Once signed in, click the Add Printer button.
Use the search bar and type BNE to find relevant printers. (You must do this for printers to appear).
Select the printer(s) you wish to install and click Add Printer.

The relevant printer(s) will now be installed and be ready for use.

### Verify & Print

To verify, go to System Settings > Printers & Scanners. The new printer(s) should appear in the list.

You can now select this printer from the print dialog in any application (Word, Chrome, etc.).

## Common Issues:

*   "Driver Unavailable": Please restart your computer. This refreshes the Universal Print service and usually resolves the error.
*   Printer not listed: Ensure you clicked the "Work or School" link in Step 3 for Windows.
*   Still stuck? Please submit a request and IT will investigate further.
